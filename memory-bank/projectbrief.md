# Project Brief: Ball Breaker

## Core Requirements

- A classic Breakout/ball-breaker clone built with Python and PyGame
- Player controls a paddle at the bottom of the screen
- Ball bounces around breaking blocks at the top
- Game ends when ball falls past the paddle

## Goals

- Functional arcade-style breakout game
- Clean, modular Python code
- Expandable for future features (screens, powerups, difficulty)

## Source of Truth

- `game.py` - Main game loop and entry point
- `models/` - Game object classes (paddle, ball, block, options)
- `docs/plan.md` - Feature planning and todo tracking
