# Product Context: Ball Breaker

## Why This Project Exists

- Personal experimentation with Python and PyGame
- Learning game development fundamentals
- A fun side project to build and expand incrementally

## Problems It Solves

- Provides a simple, playable breakout game
- Demonstrates PyGame's capabilities for 2D game development
- Serves as a sandbox for adding new features (powerups, difficulty, screens)

## How It Should Work

- Player controls a paddle using left/right arrow keys
- Ball sits on the paddle and launches with spacebar
- Ball bounces off walls, paddle, and blocks
- Blocks disappear on contact; player scores by clearing blocks
- Ball falling past the paddle = death
- Game loop runs at 60 FPS

## User Experience Goals

- Responsive paddle controls
- Satisfying ball physics
- Clear visual feedback (explosions on death)
- Expandable into full breakout experience with screens, lives, stats
